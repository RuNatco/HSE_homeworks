## Longest Common Subsequence (LCS)

### Алгоритм
Ищем длину LCS динамикой по двум индексам.
- Создаем таблицу dp размера (m+1) x (n+1), где dp[i][j] — длина LCS для string_1[:i] и string_2[:j].
- Переход:
  - если string_1[i-1] == string_2[j-1], то dp[i][j] = dp[i-1][j-1] + 1;
  - иначе dp[i][j] = max(dp[i-1][j], dp[i][j-1]).
- После заполнения восстанавливаем одну LCS, идя из dp[m][n] назад:
  - если символы равны, включаем его в ответ и идем по диагонали;
  - иначе двигаемся в сторону большего значения (up или left).

### Сложность
- Время: O(m * n), где m и n — длины строк.
- Память: O(m * n) для таблицы dp. Можно снизить память до O(min(m, n)), если нужна только длина, без восстановления последовательности.
